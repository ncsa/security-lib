What is this project?

This is a set of various extremely useful bits of code that have evolved over several projects at the NCSA's CSD.

(National Center for Supercomputing Applications, Cybersecurity Directorate).

They are widely used in projects such as OA4MP, CILogon and QDL.

[top]

Why have these?

Mostly these are specialized and do common operations that may be much more complex in other packages. For instance, the crypto module has X 509 certificate and key utilities that use native Java, so you don't have to import some huge stack of code to do them.
